# Glossary 

Terms used throughout this wiki

| Term         | Definition                                                                                                        |
|:-------------|:------------------------------------------------------------------------------------------------------------------|
| Silver       | The default currency used in-game                                                                                 |
| CPs (EMoney) | Conquer Points. This is referred to as 'EMoney' in the client.                                                    |
| CPs (Bound)  | Conquer points given to the player but cannot be traded                                                           |
| Currency     | Term for silver, CPs, or Bound CPs                                                                                |
| Packet       | A raw unit of data sent over the network between client and server                                                |
| Message      | A structured packet with a defined type and payload                                                               |
| Player       | A character you can see in the in-game world.  Many instances of players. Also refers to a human clicking buttons |
| Hero         | The logged in player's session. One instance of the hero                                                          |
